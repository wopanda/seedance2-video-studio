# Seedance 2.0 Video Studio

> 状态：V1 可用，当前已接通 APIMart / doubao-seedance-2.0，可真实创建视频任务、轮询结果并下载视频。

这是一个给**自己用**的 Seedance 2.0 视频生成 skill。

它不是纯 prompt skill，也不是剧本工作台。
它的目标是完成一条最短闭环：

1. 用户输入需求与素材
2. AI 自动判断最合适的 Seedance 2.0 模式
3. AI 自动做轻量结构化提示词扩展
4. AI 调接口发起生成
5. AI 跟踪任务并返回结果

## V1 支持范围

- Text-only：纯文字生成
- First Frame：首帧图生
- First + Last Frame：首尾帧控制
- 轻量提示词扩展
- 任务创建 / 状态查询 / 结果下载
- APIMart 图片上传后的视频生成

暂不默认支持：
- 多段自动拼接
- 长视频自动拆段
- 自动生图
- 完整剧本与分镜流水线
- 稳定的飞书自动回传闭环

## 默认交互原则

### 1. 用户只说意图
用户可以直接说：
- 做一个 8 秒竖屏视频，雨夜街头，女生慢慢回头看镜头，电影感。
- 用这张图做首帧，生成一个 6 秒视频，人物从静止到微笑转头。
- 用图1做首帧，图2做尾帧，生成一个 6 秒过渡视频。

### 2. AI 负责翻译成生成语言
AI 自动完成：
- 模式判断
- 素材映射
- 提示词结构化扩展
- 默认参数补全
- 生成执行与结果回传

### 3. 只在必要时打断
只有以下情况才询问用户：
- 多张图主次不清
- 新生成 / 延长意图冲突
- 用户明确要求“先给我看 prompt”
- 需求存在明显风格冲突

## 执行规则

### A. 先判断模式
优先按以下顺序判断：
1. 有 2 张图且明显是起止状态 → First + Last Frame
2. 有 1 张主图 → First Frame
3. 只有文字 → Text-only
4. Extend 模式后续进入 V2

### B. 做轻量提示词扩展
默认不是把句子写长，而是做结构化翻译：
- 补风格与镜头节奏
- 补时间结构（开场 / 中段 / 结尾）
- 补 negative constraints
- 补 generation settings

详细规则见：
- `references/interaction-spec-v1.md`
- `references/prompt-expansion-rules-v1.md`

### C. 优先最短成功路径
默认目标是：
- 一次输入
- 一次模式判断
- 一次生成
- 一次返回结果

不要在 V1 里引入不必要的复杂工作流。

## 输出格式

默认向用户返回：
1. 使用模式
2. 关键参数
3. task_id / 状态
4. 成功后的视频结果
5. 若失败，给出简短可执行建议

## 设计原则

- 默认简单，内部专业
- 人定方向，AI定做法
- 不把专业术语压给用户
- 先保证单次生成闭环，再持续迭代到多段与拼接

## 当前迭代约束

V1 明确允许后续调整：
- 默认时长
- 默认画幅
- 提示词扩展力度
- 确认闸门阈值
- 飞书返回样式

这不是一次写死的 skill，而是一个可持续进化的自用生成器。
